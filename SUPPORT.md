# Support

- Open issues in GitHub: https://github.com/rock4007/Janus_PQC/issues

## Before Filing an Issue
- Include OS, Python version, and steps to reproduce.
- Attach logs or outputs from `run_tests_janus_pqc.py`.
- Specify whether `oqs` is present or using Ed448 fallback.

## Security
- Do not disclose sensitive data in issues.
